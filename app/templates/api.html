get predictions

To get predictions you need to make a POST call to the following url :
http://ds4ateam12.mooo.com/api/v1/predict 


# Here is a POST request

{
    "time": 3,
    "data": [
        {
            "EdadMeses": 53.0, 
            "IdBeneficiario": 7196373, 
            "ZScoreIMC": -0.19, 
            "ZScorePesoTalla": -0.16, 
        }, 
        {
            "EdadMeses": 59.0, 
            "IdBeneficiario": 7196373, 
            "ZScoreIMC": 0.48, 
            "ZScorePesoTalla": 0.57, 
        }
    ]
}

QUERY PARAMETERS

Field Type Description
time Integer (optional) Prediction time in months, default = 3
data Object (required) A list of objects. At least two datas are required to get a prediction for one child. Details below.

DATA PARAMETERS

Field Type Description
IdBeneficiario Integer (required) Id of the child data.
EdadMeses Integer (required) Age in month of child.
ZScorePesoTalla Float (required) Weight for height Zscore 
ZScoreIMC Float (required) BMI Zscore
